Welcome on FlightR!

This application was created to learn and demonstrate the use of frameworks such as Apache Spark in web contexts. Apache Spark is a fundamental framework in the big data field and allows complex analyzes in memory (i.e. on RAM memory) on a huge dataset with milions of records.

On this page you will find an example bar chart that indicates the distribution of an attribute (entire dataset). You can choose the feature to be displayed with the red button at the top left of the graph.
